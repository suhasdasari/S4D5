# S4D5 Settlement Layer (Smart Contracts)

This directory contains the Smart Contracts that handle the settlement of trades proposed by the Alpha Strategist.

## Architecture
- **Network**: Compatible with Hedera and EVM chains.
- **Function**: Executes trade proposals via decentralized exchanges or on-chain protocols.
- **Integration**: Accessed by the `Execution Agent` (in `backend/core_agents/execution_agent`).
