# Alpha Strategist

**Role**: Orchestrator & Planner
**Tools**: LangGraph, 0G Labs SDK
**Input**: User Goal (Natural Language)
**Output**: Structured Trade Proposal

## Responsibilities
1.  **Goal Decomposition**: Breaks down high-level objectives into tasks.
2.  **Consensus**: Aggregates output from Bull and Bear analysts using weighted logic.
3.  **Audit**: Logs all decisions to the 0G Labs Data Availability layer.
