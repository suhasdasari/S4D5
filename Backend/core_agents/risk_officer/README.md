# Risk Officer

**Role**: Guardian
**Tools**: Volatility Indices, liquidity metrics
**Input**: Preliminary Trade Proposal
**Output**: Approval/Rejection with Risk Score

## Responsibilities
1.  **Sanity Check**: Ensures proposals match risk appetite.
2.  **Liquidity Audit**: Verifies sufficient market depth to execute without slippage.
3.  **Kill Switch**: Can veto the Alpha Strategist.
