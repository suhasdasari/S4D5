# Execution Agent

**Role**: Trader
**Tools**: Ethers.js, Hedera SDK
**Input**: Fully Approved Proposal
**Output**: Transaction Hash

## Responsibilities
1.  **Routing**: Selects best venue (Uniswap, SaucerSwap, etc.).
2.  **Execution**: Signs and broadcasts the transaction.
3.  **Verification**: Confirms on-chain settlement.
